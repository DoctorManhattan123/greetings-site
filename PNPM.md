# Small PNPM cheat sheet

If you have npm you can install pnpm like this:

```sh
npm i -g pnpm
```

## Install all

```sh
// npm
npm i

// pnpm
pnpm i
```

## Install dependency

```sh
// npm
npm i <deps>

// pnpm
pnpm add <deps>
```

## Install dependency

```sh
// npm
npm i -D <deps>

// pnpm
pnpm add -D <deps>
```

## Run command

```sh
// npm
npm run <command>

// pnpm
pnpm <command>
```
